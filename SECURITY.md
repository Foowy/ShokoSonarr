# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities using GitHub's private reporting flow rather than
a public issue: go to the [Security tab](https://github.com/Foowy/ShokoSonarr/security)
and select "Report a vulnerability".

This project follows the same reporting process as
[ShokoServer](https://github.com/ShokoAnime/ShokoServer/security) and
[ShokoRelay](https://github.com/natyusha/ShokoRelay/security).

## Supported Versions

Only the latest released version is supported. Please update before reporting an issue
if you're not on the latest release.
